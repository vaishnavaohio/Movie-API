myFlix App

This web application allows users to access information about different movies, directors and genres.
Users are able to sign up, update their personal information and create a list of their favorite movies.

Business Logic URL HTTP method Req Body Format Res Body Format
Get List of Movies /movies/ GET None JSON object with all movie details
Get Information on a single movie by title /movies/:title/ GET None JSON object showing data on a single movie, ex:
{
title: "",
genre: "",
description: "" ,
director:{name: bio: birth-year: death-year: } ,
image URL: "",
feature: "yes/no"
}
Get data about a genre /movies/genre/:title GET None Return description of movie by title, ex:

[title] is a [genre] film about [description]
Get information about a director by name /movies/director/:name GET None JSON object with director information
{
name: "",
bio: "",
birth-year: "",
death-year: ""
}
Allow user to register /users POST JSON object with user infromation
{
Username: [username]
Password:[password]
Email address: [email address]
} 		JSON object that returns user details
{
Username: "BMars101"
Password: "*******"
Email address: "beth.ellieday@gmail.com"
}
Allow user to update username /users/:username PUT None send text message indicating username was successfully updated ex: ":username was successfully updated"
Add a movie to movie list users/:username/movies/:movieID POST JSON object with movie details
{
title: "",
genre: "",
description: "" ,
director:{name: bio: birth-year: death-year: } ,
image URL: "",
feature: "yes/no"
} 		JSON object with the movie and details just added
{
title: "",
genre: "",
description: "" ,
director:{name: bio: birth-year: death-year: } ,
image URL: "",
feature: "yes/no"
}
Delete a movie from movie list user/:username/movies/:movieID DELETE None Text message indicating the movie was successfully removed from list.
"movie was successfully removed from list."
Allow user to deregister /users/:username DELETE None text message indicating user successfully deregistered account
":username successfully deregistered"
